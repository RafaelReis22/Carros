# Carros Java

Projeto em Java que implementa diferentes modelos de carros, cada um com características específicas de combustível, consumo e capacidade do tanque.

## Modelos Implementados

- **Esportivo**: Gasolina, 6 Km/L, 45L
- **Utilitário**: Diesel, 5 Km/L, 70L
- **SUV**: Gasolina, 8 Km/L, 55L
- **SUVFlex**: Gasolina (8 Km/L) ou Álcool (6 Km/L), 65L
- **Econo**: Gasolina, 20 Km/L (reduz 1Km/L a cada 5000Km até 10Km/L), 55L
